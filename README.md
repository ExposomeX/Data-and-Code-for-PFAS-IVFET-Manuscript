The repository includes the following files:
▪code_epidemiological_association: Scripts for epidemiological association analyses, including single-exposure models and mixture-exposure models.
▪code_bioinformatic_analysis: Scripts for bioinformatic analyses based on the Comparative Toxicogenomics Database and GeneCards, including candidate gene identification and pathway enrichment analyses.
▪code_ebio_PFAS_pcos: Scripts for biological graph network analysis using the E-BIO module to evaluate PFAS–Gene Ontology–PCOS relationships.
More information can be obtained by contacting the corresponding author: Dr. Bin Wang (binwang@pku.edu.cn)
